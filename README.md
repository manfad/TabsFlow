## ğŸš€ Quick Start

### Prerequisites

- **Node.js** (v18 or higher)
- **Rust** (latest stable)
- **pnpm** (recommended package manager)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/manfad/TabsFlow.git
   cd TabsFlow
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Run in development mode**
   ```bash
   pnpm dev
   ```

4. **Build for production**
   ```bash
   pnpm tauri build
   ```
